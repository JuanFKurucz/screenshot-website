# screenshot-website

Electron-js tool to take a full webpage screenshot

## Installation

Clone and run:

```bash
npm install
```

or

```bash
yarn
```

## Usage

```bash
npm start url=https://github.com/JuanFKurucz/
```

or

```bash
yarn start url=https://github.com/JuanFKurucz/
```
